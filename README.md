1) Как развернуть сайт
   1) Скачиваем python
   2) Скачиваем PyCharm
   3) Открываем PyCharm
   4) Открываем консоль и пишем "pip install git"
   5) Скачиваем все библиотеки из requirements.txt
   6) Запускаем main.py
   7) Переходим на 127.0.0.1:5000
